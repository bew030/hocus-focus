Hocus Focus


Hocus Focus is a Google Chrome extension project we made during the SD Hacks 2019 Annual Hackathon. We chose to create something that would help us personally as college students and wanted to learn more about web applications and extensions, which led us to the idea of creating a study tool that would be easily accessible and minimal at the same time.

It took about 24 hours and was our first time coding in Javascript and using Google Chrome's API, combined with very limited knowledge in HTML and CSS. Please be gentle and don't do anything too crazy with it, she's a bit fragile. :-)

You can find the source code and corresponding files by following this link to our Github repository and more about SD Hacks here.

- Made with love by Sebastian Dogaru, Bernard Wong and Neil Tengbumroong ♥
